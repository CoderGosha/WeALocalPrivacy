# WeALocalPrivacy

Статичные страницы Privacy Policy и Support для приложения **WeALocal**
(App Store Connect требует обе ссылки).

Хостинг — GitHub Pages из ветки `main` (Settings → Pages → Deploy from a branch → `main` / root).

URL после включения Pages:

| Поле в App Store Connect | URL |
|---|---|
| Privacy Policy URL | `https://codergosha.github.io/WeALocalPrivacy/` |
| Support URL | `https://codergosha.github.io/WeALocalPrivacy/support.html` |

Обе страницы двуязычные (русский + английский, якоря `#ru` / `#en`).

При существенном изменении политики — обновить дату «Действует с … / Effective …» в `index.html`.
